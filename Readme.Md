# 📂 DocVault

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Node.js](https://img.shields.io/badge/Node.js-Express-green)
![MongoDB](https://img.shields.io/badge/Database-MongoDB-green)
![React](https://img.shields.io/badge/Frontend-React-blue)
![Status](https://img.shields.io/badge/Status-Active-success)

DocVault is a secure and scalable **Document Management System (DMS)** inspired by **Google Drive**, built using the **MERN Stack**. It enables organizations to securely store, organize, manage, and share documents with advanced access control, version history, file sharing, and role-based permissions.

---

## ✨ Features

### 🔐 Authentication & Security
- JWT Authentication
- Role-Based Access Control (RBAC)
- Secure Password Encryption
- Protected Routes
- Permission-Based APIs

### 📁 Document Management
- Upload Documents
- Download Files
- File Preview
- Create Nested Folders
- Move & Organize Files
- Search Documents
- Filter & Sort Files
- Recycle Bin
- Permanent Delete

### 📝 Version Control
- Automatic Version History
- View Previous Versions
- Restore Older Versions

### 🤝 Sharing
- Share Files & Folders
- Generate Public Share Links
- Permission-Based Sharing
- Owner & Shared Document Management

### 👥 User Management
- Create Users
- Manage Roles
- Department Management
- Project Management
- User Profiles

### 📊 Dashboard
- Storage Overview
- Recent Documents
- User Statistics
- Activity Tracking

### 🔔 Notifications
- User Notifications
- Document Updates
- Sharing Alerts

### 🛡️ Security
- Password Hashing
- JWT Authorization
- Secure API Access
- Audit Logging

# 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/Codewithakk/DocVault.git
```

### 2. Navigate to the project

```bash
cd DocVault
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

The application will now be running locally.
